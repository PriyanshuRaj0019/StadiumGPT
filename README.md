# 🏟️ StadiumGPT — Smart Stadium Operations Assistant
**A next-generation Generative AI operational ecosystem built for the FIFA World Cup 2026.**

**StadiumGPT** was built as a submission for the **Hacker2Skill PromptWars Hackathon**. It moves beyond standard chatbots to solve complex, real-world stadium logistics for fans, staff, and organizers using Retrieval-Augmented Generation (RAG) and Agentic AI workflows.

---

## ✨ Key Features

- 🌍 **Multilingual Fan Assistant:** A highly localized RAG pipeline that answers specific queries about gates, seating, allowed items, and rules in multiple languages instantly.
- 🗺️ **Smart Crowd Navigation:** Interactive mapping (Leaflet.js) combined with AI-driven routing to steer fans away from bottlenecked concourses in real-time.
- 📊 **Crowd Intelligence Dashboard:** Real-time stadium telemetry visualized with Chart.js, featuring an "AI Operations Copilot" that analyzes queue density and suggests tactical staff deployments.
- 🚨 **Volunteer Copilot:** One-click GenAI generation of strict, contextual Standard Operating Procedures (SOPs) for security alerts, medical emergencies, or lost children.
- ♿ **Accessibility Support:** Dedicated AI routing and assistance for wheelchair users and fans with sensory sensitivities.

---

## 🛠️ Tech Stack

**Frontend (The Face):**
- HTML5, CSS3 (Custom Glassmorphism UI)
- Vanilla JavaScript
- Chart.js (Data Visualization)
- Leaflet.js (Interactive Mapping)

**Backend (The Brain):**
- Python 3 & FastAPI
- Google Gemini 2.5 Flash API
- LangChain & HuggingFace Embeddings
- FAISS (Vector Database for RAG)
